# TicTacToe
Tic Tac Toe Game for Android Phones

    
**Demo:**
[![TicTacToe Applicaiton](https://user-images.githubusercontent.com/90371913/152211721-7eb596a0-8465-4ea8-9d10-8991593471a0.jpg)](https://youtu.be/-XKqe9O-2Bg)  


Using **Kotlin**  

This Code Provide Simple **Game** For **Android Users**

This Application Called **Tic Tac Toe**
&nbsp;  
&nbsp;  

**This Game Will:**
* • Make the player Love Tic Tac Toe game
* •	Keep Scoring for the winner
* •	Announce the winner
* •	Announce the loser
* •	Announce for draw
&nbsp;  
&nbsp;  

**This Game Contains 3 Playing Options:** 

**1. No Plyer Option:**
*     •	Game Between two computers
*     •	The first computer will represent the player
*     •	The player will start the game by choosing the first square 
&nbsp;  
**2. One Player Option:**
*     •	Game Between Player and computer
*     •	The player will start the game by choosing the first square 
&nbsp;  
**3. Two Players Option:**
*     •	Game Between Two Players
*     •	The red color will represent the first player
*     •	The green color will represent the second player
*     •	The red color will start the game
